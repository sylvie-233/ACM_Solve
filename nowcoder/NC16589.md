# 机器翻译



字符串构造模拟
```python
m, n = list(map(int, input().split()))
arr = input().split()
st = set()
lt = []
ans = 0

for i, t in enumerate(arr):
    if t in st:
        continue
    lt.append(t)
    st.add(t)
    if len(lt) > m:
        item = lt[0]
        lt = lt[1:]
        st.remove(item)
    ans += 1
print(ans)
```
