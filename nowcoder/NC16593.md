# 铺地毯


枚举
```python
n = int(input())
arr = []
for _ in range(n):
    arr.append(list(map(int, input().split())))

x1, y1 = list(map(int, input().split()))
ans = -1

for i, t in enumerate(arr):
    x, y, g, k = t
    if x <= x1 and x1 <= x + g and y <= y1 and y1 <= y + k:
        ans = i + 1

print(ans)
```
