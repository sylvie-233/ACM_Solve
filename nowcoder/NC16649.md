# 校门外的树


一维重叠区域计算，对l、r进行排序计数
```python
L, M = list(map(int, input().split()))
arr: list[tuple[int, int]] = []
for _ in range(M):
    l, r = list(map(int, input().split()))
    arr.append((l, 0))
    arr.append((r, 1))

# 对l、r进行排序
arr.sort()

sum, cnt, start = 0, 0, -1
for i in arr:
    p, f = i
    if start == -1:
        start = p
    # 遇到l，计数+1
    if f == 0:
        cnt += 1
    # 遇到r，计数-1
    else:
        cnt -= 1
        # 计数减到0，则为一个完整的重叠区域
        if cnt == 0:
            # 累计重叠区域的长度
            sum += p - start + 1
            start = -1

print(1 + L - sum)
```
